# Telegram AI Bot

Многофункциональный бот для Telegram с поддержкой различных AI-моделей через ProxyAPI.

## ✨ Возможности

- **Множество моделей:**
  - ⭐ **OpenAI:** GPT-5.2, GPT-5 Mini, GPT-5.1 Codex
  - 🎭 **Claude:** Opus 4.5, Sonnet 4.5, Haiku 4.5
  - 🧠 **DeepSeek:** V3, Reasoner
  - ⚡ **Google:** Gemini 2.5 Flash
  - 🐉 **Qwen:** Max, Coder Plus
  - 🔧 **Другие:** GLM 4.6, Grok 3

- 🎯 **Три режима ответов:**
  - 🟢 Краткий - только суть (512 токенов)
  - 🟡 Обычный - сбалансированно (1024 токена)
  - 🔴 Подробный - развернутые ответы (2048 токенов)

- 🎛 **Удобное меню с inline-кнопками**
- 💾 **Сохранение истории в SQLite**
- ⚙️ **Индивидуальные настройки для каждого пользователя**
